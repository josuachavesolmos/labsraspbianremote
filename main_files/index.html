Objetivo

Diseñar, implementar y probar una mini-plataforma de dos laboratorios remotos, autónomos, que tengan aplicativo tipo página web, para el área de circuitos eléctricos y/o electrónicos.

Saber mas.

Plantas de Control

Dos plantas, enfocadas en el área de control electrónico de los laboratorios remotos, temperatura y distancia.

AD/DA

Un módulo para el control y comunicación entre las plantas electrónicas y el aplicativo web.

Aplicación Web

Aplicativo página web para el acceso e interacción del usuario con las plantas de laboratorios remotos.

¿Eres novato?

Accede y descubre prácticas de laboratorio para estudiantes de primeros semestres en ingeniería electrónica, conoce en vivo como funciona.
